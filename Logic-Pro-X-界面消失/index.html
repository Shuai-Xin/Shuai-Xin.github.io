Logic Pro X 界面消失：新手最常見的問題，一秒解決。

界面消失

遙想當年，同時身為 Mac 電腦萌新、Logic 菜雞的我，也曾遇過這個問題：

  • Logic Pro 的主要視窗怎麼突然消失了？

身為一個小機靈鬼，第一個動作當然就是直接給他 command + q 關掉、再重開一次咯！

可惜，這招並沒有作用。

解決方案

一秒解決：

  • 在左上角程式顯示為 Logic Pro 時，使用 command + 1

完畢！

原理解釋

  • 在 Mac OS 的 GUI 邏輯中，一個應用程式（Application），可以有著多個不同的視窗（Windows）。
  • 以 Logic Pro X 為例，它就分別有：
    • Main Window（預設開啟快捷鍵為 command + 1）
    • Mixer
    • Smart Controls
    • Piano Roll
    • Score Editor
    • Audio File Editor
    • Event List
    • Project Audio
    • MIDI Transform
    • MIDI Environment
  • 而新手看習慣了第一個視窗：Main Window。所以當它被無意關掉後，第一個直覺就是「怎麼消失了」。
  • 而重啟無效的原因是，Logic 會記得你上次關閉時有開啟的視窗，下次也顯示相同的視窗組合。所以當然也不會因為重啟而幫你打開咯。
