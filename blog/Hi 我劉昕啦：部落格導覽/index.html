發表於2022-03-05|更新於2022-03-23|架站筆記
|閱讀量:

這裡是哪？

  • 這裡是劉昕的部落格，會放一些自己抓歌的和弦筆記、家教時遇到值得記錄或是備忘的內容、音樂製作小撇步分享（DAW 用的是 Logic Pro X）。感興趣可以到 文章分類 翻找看看。
  • 這裡有淺色與深色兩種主題可以更換，從右下角的齒輪中設定（手機瀏覽則需往下滑一小段距離才會顯示出來）。
  • 為了保持簡約，這裡採用的評論系統是 Utterances ，需要有 github 賬號才能發言，還請見諒。
